## Prerequisites
* Kubernetes 1.19+
* [Helm](https://github.com/helm/helm/releases) 3.8.0+

## TL;DR

```bash
helm repo add stevenjdh https://StevenJDH.github.io/helm-charts
helm repo update
helm search repo stevenjdh
helm install my-release stevenjdh/<chart>
```
